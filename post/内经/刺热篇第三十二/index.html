刺热篇第三十二

刺热

📜 原文

{% note primary %} 　　肝热病者，小便先黄，腹痛多卧身热，热争则狂言及惊，胁满痛，手足躁，不得安卧，庚辛甚，甲乙大汗，气逆则庚辛死，刺足厥阴少阳，其逆则头痛员员，脉引冲头也。心热病者，先不乐，数日乃热，热争则卒心痛，烦闷善呕，头痛面赤无汗，壬癸甚，丙丁大汗，气逆则壬癸死，刺手少阴太阳。脾热病者，先头重颊痛，烦心颜青，欲呕身热，热争则腰痛不可用俯仰，腹满泄，两颔痛，甲乙甚，戊己大汗，气逆则甲乙死，刺足太阴、阳明。肺热病者，先淅然厥，起毫毛，恶风寒，舌上黄身热。热争则喘咳，痛走胸膺背，不得大息，头痛不堪，汗出而寒，丙丁甚，庚辛大汗，气逆则丙丁死，刺手太阴阳明，出血如大豆，立已。肾热病者，先腰痛䯒酸，苦渴数饮，身热，热争则项痛而强，䯒寒且酸，足下热，不欲言，其逆则项痛员员澹澹然，戊己甚，壬癸大汗，气逆则戊己死，刺足少阴太阳，诸汗者，至其所胜日汗出也。肝热病者，左颊先赤，心热病者颜先赤，脾热病者鼻先赤，肺热病者右颊先赤，肾热病者颐先赤，病虽未发，见赤色者刺之，名曰治未病。热病从部所起者，至期而已；其刺之反者，三周而已；重逆则死。诸当汗者，至其所胜日，汗大出也。

　　诸治热病，以饮之寒水乃刺之，必寒衣之，居止寒处，身寒而止也。热病先胸胁痛，手足躁，刺足少阳，补足太阴，病甚者为五十九刺。热病始手臂痛者，刺手阳明太阴而汗出止。热病始于头首者，刺项太阳而汗出止。热病始于足胫者，刺足阳明而汗出止。热病先身重骨痛，耳聋好瞑，刺足少阴，病甚为五十九刺。热病先眩冒而热，胸胁满，刺足少阴少阳。太阳之脉，色荣颧骨，热病也，荣未交，曰今且得汗，待时而已。与厥阴脉争见者，死期不过三日，其热病内连肾，少阳之脉色也。少阳之脉，色荣颊前，热病也，荣未交，曰今且得汗，待时而已，与少阴脉争见者，死期不过三日。热病气穴：三椎下间主胸中热，四椎下间主鬲中热，五椎下间主肝热，六椎下间主脾热，七椎下间主肾热，荣在骶也。项上三椎陷者中也。颊下逆颧为大瘕，下牙车为腹满，颧后为胁痛，颊上者鬲上也。 {% endnote %}

🌿 白话文

{% note success %} 关于五脏热病的症状与治疗：

肝脏热病：患者最初会小便发黄，接着出现腹痛、嗜睡、身体发热。当热邪加剧时，会出现胡言乱语、惊悸、胁部胀痛、手脚躁动、难以安卧。病情在庚辛日（金旺之日）加重，到甲乙日（木旺之日）会因出汗而缓解；若气机上逆，可能在庚辛日死亡。治疗应针刺足厥阴肝经和足少阳胆经。若气逆上行，则会出现持续性头痛，这是热邪沿经脉上冲头部的表现。

心脏热病：患者最初情绪低落，数日后开始发热。热邪加剧时会突发心痛、烦躁胸闷、频繁呕吐、头痛面红且无汗。病情在壬癸日（水旺之日）加重，到丙丁日（火旺之日）会因出汗缓解；气逆者可能在壬癸日死亡。治疗应针刺手少阴心经和手太阳小肠经。

脾脏热病：患者最初头重、面颊痛、心烦、脸色发青、想呕吐、身体发热。热邪加剧时会出现腰痛难弯腰、腹胀腹泻、下颌疼痛。病情在甲乙日（木旺之日）加重，到戊己日（土旺之日）会因出汗缓解；气逆者可能在甲乙日死亡。治疗应针刺足太阴脾经和足阳明胃经。

肺脏热病：患者最初突然发冷、汗毛竖起、怕风寒，舌苔发黄后出现发热。热邪加剧时会出现咳嗽气喘、胸背疼痛、呼吸不畅、剧烈头痛、出汗后发冷。病情在丙丁日（火旺之日）加重，到庚辛日（金旺之日）会因出汗缓解；气逆者可能在丙丁日死亡。治疗应针刺手太阴肺经和手阳明大肠经，放出豆粒大小的血即可见效。

肾脏热病：患者最初腰痛腿酸、口渴多饮、身体发热。热邪加剧时会出现颈项僵痛、小腿发冷酸胀、脚底发热、沉默寡言。若气逆上行，则颈项持续疼痛并伴有眩晕感。病情在戊己日（土旺之日）加重，到壬癸日（水旺之日）会因出汗缓解；气逆者可能在戊己日死亡。治疗应针刺足少阴肾经和足太阳膀胱经。所有需要通过发汗治疗的热病，在对应脏腑旺盛的日子进行治疗，汗出后病情就会好转。

面部预兆：
肝热病患者左颊先发红，心热病者额头先发红，脾热病者鼻部先发红，肺热病者右颊先发红，肾热病者下巴先发红。即使疾病尚未发作，只要出现对应部位发红，就应及时针刺治疗，这称为"治未病"。

治疗原则：
热病从特定部位开始发作的，及时治疗可痊愈；若误用针刺，需三周才能恢复；多次误治会导致死亡。所有需发汗的热病，在对应脏腑旺盛之日治疗可大量排汗而愈。

其他治疗细则：

  1. 治疗热病前应先让患者喝凉水，针刺时让患者穿薄衣、处于阴凉处，直至体温下降。

  2. 胸胁痛伴手脚躁动者：刺足少阳胆经，补足太阴脾经；重症用"五十九刺法"。

  3. 手臂先痛者：刺手阳明大肠经和手太阴肺经至出汗。

  4. 头部先发病者：刺项部足太阳膀胱经至出汗。

  5. 小腿先发病者：刺足阳明胃经至出汗。

  6. 身重骨痛、耳聋嗜睡者：刺足少阴肾经，重症用五十九刺法。

  7. 头晕发热伴胸胁胀满者：刺足少阴肾经和足少阳胆经。

面诊与预后：

  • 颧骨发红属太阳经热病，若红色未扩散，及时发汗可愈；若与厥阴经症状同现，三日内可能死亡（热邪已深入肾脏）。

  • 面颊前部发红属少阳经热病，红色未扩散可发汗而愈；若与少阴经症状同现，三日内可能死亡。

热病对应穴位：
第三椎下（身柱穴）主胸中热，第四椎下（无穴）主膈中热，第五椎下（神道穴）主肝热，第六椎下（灵台穴）主脾热，第七椎下（至阳穴）主肾热。面诊对应：颊下红色延至颧骨为腹内肿块，下颌部红为腹胀，颧骨后方红为胁痛，颊上部红为膈上病变。 {% endnote %}

二、热病面诊与"治未病"

面部对应：

  • 左颊（肝）→颜（心）→鼻（脾）→右颊（肺）→颐（肾）

防治原则：

  • 见对应部位发赤，虽未发病即行针刺，此为"治未病"。

三、热病起始部位刺法

  • 胸胁痛、手足躁：刺足少阳胆经，补足太阴脾经（重症用"五十九刺"）
  • 始于手臂痛：刺手阳明大肠经、手太阴肺经
  • 始于头部：刺足太阳膀胱经项部
  • 始于足胫：刺足阳明胃经
  • 身重骨痛、耳聋嗜睡：刺足少阴肾经（重症用"五十九刺"）
  • 眩晕发热、胸胁满：刺足少阴肾经、足少阳胆经

四、热病气穴定位

椎间定位 对应病位
第三椎 胸中热
第四椎 膈中热
第五椎 肝热
第六椎 脾热
第七椎 肾热
骶部 荣血之热
项上三椎 定位取穴标志点

五、面诊病理征象

  • 颊色逆传至颧骨：提示大腹瘕积
  • 下颌部色变：提示腹胀
  • 颧后色变：提示胁痛
  • 颊上部色变：提示膈上病变

六、预后判断

汗出规律：

  • 在本脏旺日（所胜日）可汗出而解。

危重指征：

  • 太阳经热病：见颧骨发赤，若未扩散可待时自愈；若与厥阴脉证并见，三日内危殆。
  • 少阳经热病：见颊前发赤，未扩散可自愈；若与少阴脉证并见，三日内危殆。

七、治疗总则

降温三部曲：

  1. 先饮寒水
  2. 再行针刺
  3. 更衣薄服
  4. 居处阴凉，至身凉为度

针刺禁忌：

  • 误刺需三周恢复，反复误治则致死。

此整理以症状-时间-治疗三维框架呈现，突出中医"天人相应"思想与"治未病"理念，保留关键诊断要点与治疗规范。

刺志论篇第五十三

刺志论

📜 原文

{% note primary %} 　　黄帝问曰：愿闻虚实之要。岐伯对曰：气实形实，气虚形虚，此其常也，反此者病。谷盛气盛，谷虚气虚，此其常也，反此者病。脉实血实，脉虚血虚，此其常也，反此者病。帝曰：如何而反？岐伯曰：气虚身热，此谓反也。谷入多而气少，此谓反也。谷不入而气多，此谓反也。脉盛血少，此谓反也。脉少血多，此谓反也。气盛身寒，得之伤寒。气虚身热，得之伤暑。谷入多而气少者，得之有所脱血，湿居下也。谷入少而气多者，邪在胃及与肺也。脉小血多者，饮中热也。脉大血少者，脉有风气，水浆不入，此之谓也。夫实者，气入也。虚者，气出也。气实者，热也。气虚者，寒也。入实者，左手开针空也。入虚者，左手闭针空也。 {% endnote %}

🌿 白话文

{% note success %} 黄帝问道：我想了解关于虚实的关键原理。岐伯回答：元气充足则形体健壮，元气虚弱则形体羸弱，这是正常状态；若出现相反状况就是病态。饮食丰盛则元气充沛，饮食匮乏则元气不足，这是常态；与之相悖即为病症。脉象充盈则血液充沛，脉象虚弱则血液不足，这是常规现象；若有异常则代表疾病。

黄帝追问：怎样的表现属于反常？岐伯解释道：元气虚弱却身体发热，这是反常现象。进食量多但元气不足，属于异常。不进食反而元气过剩，同样反常。脉象强盛但血液稀少，或脉象微弱却血液充盈，这些都是病态表现。

具体而言：

  • 元气旺盛却身体发冷，源于寒邪侵袭

  • 元气不足而身体燥热，起因于暑热损伤

  • 进食量大但元气匮乏，常见于失血过多或湿邪积聚下焦

  • 进食量少却元气过剩，提示邪气侵扰胃与肺部

  • 脉象细弱但血液过盛，多因热性酒饮引发

  • 脉象洪大但血液不足，常伴随风邪入侵且无法正常饮食

总结虚实本质：

  • 实证源于邪气侵入

  • 虚证因正气外泄导致

  • 气实者多现热证

  • 气虚者多现寒证

针灸治疗法则：

  • 治疗实证时，下针后左手要开放针孔（泄邪）

  • 治疗虚证时，下针后左手需闭合针孔（固本） {% endnote %}

🌍 整理总结

{% note info %}

一、虚实常态规律

气与形

  • 正常：气盛则形体充实，气虚则形体虚弱。
  • 反常：气虚者反见身热（如中暑），气盛者反见身寒（如伤寒）。

饮食（谷）与气

  • 正常：饮食充足则气足，饮食不足则气虚。
  • 反常：
    • 食多而气少（因失血或湿邪聚于下焦）。
    • 食少而气多（邪气滞留胃肺，气机壅滞）。

脉与血

  • 正常：脉充实则血足，脉虚弱则血亏。
  • 反常：
    • 脉盛而血少（外感风邪且水饮不入）。
    • 脉弱而血多（体内湿热郁结）。

二、虚实本质与表现

  • 实症：外邪入侵，气聚于内 → 热证。
  • 虚症：正气外泄，气耗于外 → 寒证。

三、针刺治疗原则

  • 实症：刺后开大针孔，泄邪气。
  • 虚症：刺后闭合针孔，防正气外泄。

反常病理机制归纳

反常现象 病因病机
气虚身热 暑热耗气（伤暑）
气盛身寒 寒邪束表（伤寒）
谷多气少 失血或下焦湿滞，气化不足
谷少气多 胃肺邪阻，气机壅滞
脉盛（大）血少 风邪侵袭，水谷不纳
脉弱（小）血多 湿热内蕴，血行壅滞

核心思想

人体气血、饮食、脉象需协调平衡，反常状态提示病机，治疗需辨明虚实，针对性调治。

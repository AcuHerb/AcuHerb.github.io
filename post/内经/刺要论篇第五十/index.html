刺要论篇第五十

刺要论

📜 原文

{% note primary %} 　　黄帝问曰：愿闻刺要。岐伯对曰：病有浮沉，刺有浅深，各至其理，无过其道。过之则内伤，不及则生外壅，壅则邪从之。浅深不得，反为大贼，内动五藏，后生大病。故曰：病有在毫毛腠理者，有在皮肤者，有在肌肉者，有在脉者，有在筋者，有在骨者，有在髓者。是故刺毫毛腠理无伤皮，皮伤则内动肺，肺动则秋病温疟，溯溯然寒栗。刺皮无伤肉，肉伤则内动脾，脾动则七十二日四季之月，病腹胀烦不嗜食。刺肉无伤脉，脉伤则内动心，心动则夏病心痛。刺脉无伤筋，筋伤则内动肝，肝动则春病热而筋弛。刺筋无伤骨，骨伤则内动肾，肾动则冬病胀腰痛。刺骨无伤髓，髓伤则销铄胻酸，体解㑊然不去矣。 {% endnote %}

🌿 白话文

{% note success %} 黄帝问道：我想了解针刺治疗的关键要领。岐伯回答：疾病有表里深浅之分，针刺也需掌握深浅尺度。治疗时要根据病灶层次精准下针，既不可过度深入，也不能停留过浅。刺得过深会损伤内部脏器，刺得过浅则会导致邪气壅滞体表。若深浅把握不当，反而会造成严重伤害，扰动五脏机能，最终引发重大疾患。

具体而言，病灶所在可分七个层次： 第一层在体表毫毛与肌肤纹理之间，第二层在皮肤表层，第三层在肌肉组织，第四层在经脉系统，第五层在筋腱结构，第六层在骨骼系统，第七层在骨髓深处。

对应各层次的针刺要领是： 刺毫毛腠理时不可伤及真皮层，否则会扰动肺脏功能，导致秋季易发寒热交替的温疟，出现持续性寒战； 刺皮肤层时不可伤及肌肉，否则会扰动脾脏功能，在四季末的七十二天（脾土主令期）易发腹部胀满、心烦厌食； 刺肌肉层时不可伤及血脉，否则会扰动心脏功能，导致夏季易发心绞痛； 刺血脉时不可伤及筋腱，否则会扰动肝脏功能，导致春季易发高热伴随筋肉松弛； 刺筋腱时不可伤及骨骼，否则会扰动肾脏功能，导致冬季易发腹胀腰痛； 刺骨骼时不可伤及骨髓，若骨髓受损会导致小腿酸软无力，身体消瘦倦怠，行动困难。 {% endnote %}

🌍 整理总结

{% note info %}

一、刺法总原则

原则 描述
深浅适度 针刺深浅需根据病位深浅调整，遵循“各至其理，无过其道”。
过深 损伤内脏，引发大病。
过浅 导致外邪壅滞，反助邪气入侵。

二、病位层次与刺法要点

病位层次 描述
毫毛腠理 误伤皮→内动肺→秋发温疟（寒战发热）。
皮肤 误伤肉→内动脾→四季末（每季最后18日）腹胀、厌食。
肌肉 误伤脉→内动心→夏发心痛。
脉 误伤筋→内动肝→春发热病、筋脉松弛。
筋 误伤骨→内动肾→冬发腹胀、腰痛。
骨 误伤髓→髓虚消瘦、小腿酸软、全身倦怠（“体解㑊然不去”）。

三、错误刺法的核心危害

危害 描述
扰动五脏 误刺伤及深层组织，直接扰动对应脏腑功能。
时令发病 脏腑受损后，在相应季节（如肝主春、心主夏）引发特定疾病。

四、核心思想

核心思想 描述
因位施针 针刺需精准判断病位层次，遵循天人相应规律，避免过度或不及，以调和气血、祛邪护正。

（注：本篇强调中医针刺治疗中“层次对应”与“整体观”的重要性，体现了《内经》“治未病”“因时制宜”的核心理念。）
